# fMRI Quality Control Report for the NAPPY Project 


## Reference

Orchard, E, R., Voigt, K., Chopra, S., Rana, T., Ward, P., Egan, G., Jamadar,S. (2022). [The maternal brain is more flexible and responsive at rest: effective connectivity of the parental caregiving network in postpartum mothers]([https://www.medrxiv.org/content/10.1101/2022.01.11.22268989v3](https://www.biorxiv.org/content/10.1101/2022.09.26.509524v1)). bioRxiv.

------------------------------------------------------------------------

This repo contains a quality control report for  Multi-echo+Multi-band (MB+ME) resting-state fMRI data from the Neual Adaptations in the Post-Partum Year (NAPPY) project (40 first-time mothers at one-year postpartum and 39 age- and education-matched women who have never been pregnant). Rendered HTML report of data quality can be downloaded [here](https://www.dl.dropboxusercontent.com/s/dn4yz1c94bl7vj8/report_git.html?dl=0). This document is ~700mb can take up to 2 min to fully load. **Please wait** until nine "tabs" have appeared at the top of the report exploring the document.
